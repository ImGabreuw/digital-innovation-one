# Hamcrest e Matcher

### Hamcrest

* É um framework para teste unitários

* Uso de predicates existentes para fazer **asserts**

### Matcher

* Object Matcher

* Bean Matcher

* Collection Matcher

* Number Matcher

* Text Matcher

### AssertJ

* Permite escrever **Asserts** mais fluentes, ou seja, de mais fácil compreensão
