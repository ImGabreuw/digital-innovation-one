### Asserts

* São métodos utilitários usados para determinar se um caso de teste teve foi bem sucedido ou não

* Uso

  * `Assert.assertEquals(...)`

  * Importe estático do método (+ legível)

    ```java
    import static org.junit.Assert.*;

    ...

    assertEquals(...)
    ```
